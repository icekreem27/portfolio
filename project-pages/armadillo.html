Masters Project
ARMADILLO: Augmented Reality Machine-Assisted Detection and Inference in Laparoscopic Liver Operations
Project Details / Background
This masters project focuses on developing an augmented reality system to assist surgeons during laparoscopic liver operations. The ARMADILLO system combines computer vision, machine learning, and augmented reality technologies to provide real-time guidance and detection capabilities.
The project involves developing algorithms for anatomical landmark segmentation in 3D liver models, implementing deep learning models for intraoperative segmentation, and creating an augmented reality interface for surgical guidance. The system aims to improve surgical precision and patient outcomes by providing enhanced visualization and decision support.
Key components include 3D liver model processing, real-time segmentation algorithms, augmented reality visualization, and integration with laparoscopic surgical equipment. The project represents a significant advancement in computer-assisted surgery and medical imaging technology.
